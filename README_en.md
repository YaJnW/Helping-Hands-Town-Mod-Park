# Presentation
A Minecraft integration pack for Helping Hands Town server users  
For people who haven't downloaded 1.18.2 and don't want to spend ~~no~~ time looking for mods ~~lazy people~~ groups

# Demand
Computer with java 17 ([ Oracle ](http://https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html))  
Minecraft launcher that can import server update standard integration packs / MCBBS standard integration packs
